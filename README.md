# myBlog

тестовый проект по написанию блога

## Начало работы

для запуска проекта нужно прописать в application.properties
image.path=(путь для хранения картинок в файловой системе).

собрать проект и скопировать файл проекта "myblog.war" в tomcat\webapps

запустить postgresql через docker-compose.yml

прописать JAVA_HOME с версией не ниже той, с которой собирался проект

запустить tomcat

приложение будет доступно по адресу http://localhost:8080/myblog

## Запуск тестов
для запуска тестов нужно прописать в test-application.properties
image.path=(путь для хранения картинок в файловой системе)
