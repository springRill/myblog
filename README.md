# myBlog

тестовый проект по написанию блога с использованием spring boot

## Начало работы

для запуска проекта нужно прописать в application.properties
image.path=(путь для хранения картинок в файловой системе).

установить postgresql, и создать базу данных "test_myblog"

проект собирается с помощью gradle

для запуска из IDE используем bootRun

для сборки исполняемого файла используем bootJar
для запуска исполняемого файла, набираем в консоли java -jar myblog-v2.0.jar

приложение будет доступно по адресу http://localhost:8080/myblog

## Запуск тестов
для запуска тестов нужно прописать в test-application.properties
image.path=(путь для хранения картинок в файловой системе)
